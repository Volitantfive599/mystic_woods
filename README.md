# Mystic Woods

Mystic Woods is a game prototype meant to simulate a 3rd person fantasy RPG.

## Overview
This project was originally created earlier in my development journey and is being revisited to improve documentation and project structure.

## Features
- Player movement and navigation
- Finite-State-Machine based Player and Enemies
- Enemy AI
- UI Features: Mini-map, health bar

## Status
Prototype / archived project being documented for portfolio purposes.
